# Data Dictionary - BancoAgricolaRD

This data dictionary documents the relational database objects used in the Banco Agricola RD final project.

Database:

```text
BancoAgricolaRD
```

Engine:

```text
Microsoft SQL Server
```

## 1. Table Overview

| Table | Type | Purpose |
| --- | --- | --- |
| `Periodo` | Dimension | Stores year, month, month name, and normalized period date. |
| `Sucursal` | Dimension | Stores Banco Agricola branch names and assigned regions. |
| `Destino` | Dimension | Stores agricultural financing destinations, rubros, and operation types. |
| `FuenteDato` | Dimension / lineage | Stores source dataset names, file names, and descriptions. |
| `FactCarteraPrestamo` | Fact | Stores agricultural loan portfolio value and loan count by branch and period. |
| `FactAreaFinanciada` | Fact | Stores financed land tasks, loan count, and value by branch and period. |
| `FactDesembolsoCobro` | Fact | Stores disbursements, collections, and net balance by branch and period. |
| `FactMontoDestino` | Fact | Stores granted amounts, operations, tasks, and beneficiaries by destination and period. |
| `AuditoriaOperacion` | Control | Stores audit records generated by trigger activity. |
| `CarteraPrestamoHistorico` | Control | Stores previous portfolio values when portfolio records are updated. |

## 5. Views

| View | Purpose |
| --- | --- |
| `VW_CarteraPorSucursalAnio` | Aggregates loan portfolio value and loan count by branch, region, and year. |
| `VW_DesembolsosVsCobrosMensual` | Aggregates monthly disbursements, collections, and net balance. |
| `VW_DestinosTopFinanciamiento` | Aggregates financing values, operations, and beneficiaries by destination and year. |
| `VW_AreasFinanciadasSucursal` | Aggregates financed tasks, loans, and value by branch and year. |

## 6. Stored Procedures

| Procedure | Purpose |
| --- | --- |
| `SP_Resumen_Cartera_Por_Anio` | Returns total portfolio and loan count for one year. |
| `SP_TopSucursales_Cartera` | Returns top branches by portfolio value, optionally filtered by year. |
| `SP_DesembolsosCobros_PorSucursal` | Returns disbursement and collection records for one branch. |
| `SP_TopDestinos_Financiados` | Returns top financing destinations, optionally filtered by year. |
| `SP_Actualizar_CarteraValor` | Updates one portfolio value in a controlled way. |
| `SP_Indicadores_Periodo` | Returns period-level indicators across all facts. |
